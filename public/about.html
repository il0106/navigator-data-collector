Наша миссия

Navigator Data Collector - это образовательный проект, демонстрирующий возможности современного веб-браузера по сбору информации о пользовательском устройстве. Мы показываем, как работает объект navigator и какие данные можно получить с его помощью.

Технологии

Наш проект построен на современных веб-технологиях:

  • Node.js - серверная часть
  • Express.js - веб-фреймворк
  • HTML5 - структура страниц
  • CSS3 - стилизация и анимации
  • JavaScript ES6+ - клиентская логика
  • Navigator API - сбор данных браузера

Безопасность

Мы заботимся о безопасности и приватности пользователей:

  • Собираем только общедоступную информацию
  • Не запрашиваем личные данные
  • Не используем cookies для отслеживания
  • Все данные собираются только с вашего согласия
  • Информация используется только в образовательных целях

Команда

Проект создан для демонстрации возможностей веб-разработки и обучения работе с браузерными API. Мы стремимся показать, как современные веб-технологии могут быть использованы для создания интерактивных и информативных приложений.
