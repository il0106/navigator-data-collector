Добро пожаловать в Navigator Data Collector

Современный веб-сайт для сбора и анализа данных браузера

Откройте консоль разработчика (F12) чтобы увидеть собранные данные navigator

Что мы делаем

Наш сайт автоматически собирает информацию о вашем браузере и устройстве через объект navigator. Все данные выводятся в консоль браузера и отправляются на сервер для анализа.

Собираемые данные включают:

  • Информацию о браузере и операционной системе
  • Характеристики экрана и окна
  • Системные возможности (память, процессор)
  • Доступные API браузера
  • Информацию о сетевом соединении
  • Временную зону и локаль

Как использовать

1. Откройте консоль разработчика (F12 или Ctrl+Shift+I)

2. Перейдите на любую страницу сайта

3. Данные автоматически соберутся и выведутся в консоль

4. Используйте кнопку "Собрать данные navigator" для повторного сбора
